Summary

The purpose of this project was to analyse the relationship between weather and proximity to the equator.


To do this, we first used random coordinates to generate a list of over 500 cities. We then used the OpenWeatherMap API to pull weather data for each location.


Matplotlib was used to plot the relationship between latitude and certain weather aspects. The factors we examined include (1) maximum temperature, (2) humidity, (3) cloudiness, and (4) wind speed. This site provides the source data and visualisations of the data, as well as descriptions for any trends observed.
