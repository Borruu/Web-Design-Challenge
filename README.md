# Web-Design
This repository contains code for a seven-page website.
### Directory
- ***Resources*** folder contains the original data source - `cities.csv` - and python code to read data into a table - `cities.py`.
- ***Assets*** folder contains the css and images used for this site.
- ***Visualisations*** folder contains html files for four aspects of weather data.
